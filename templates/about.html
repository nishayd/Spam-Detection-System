Alogrithms

Multinomial Naive Bayes

Multinomial Navie Bayes (MNB) is a classification technique used for a text data. It assumes that features (like words) are generated from different classes (like spam or not) and calculates the probability of each class given the features. It's based on Bayes' theorem and works well with large datasets,especially in task like email filtering or document categorization.

SVC

The Support Vector Classifier (SVC) algorithm is a machine learning method used for classification tasks. It finds the best boundary (or "hyperplane") that separates different classes in the data, aiming to maximize the margin between them. This helps it make accurate predicitions even with complex and high-dimensional data.

Bernoulli Navie Bayes

The Bernoulli NB algorithm classifies data by considering whether features are present or absent. It calculates class probabilities based on these binary features, making it useful for tasks like spam detection.

Random Forest Classifier

A Random Forest Classifier combines multiple decision trees to improve prediction accuracy. Each tree is trained on a random subset of the data and a features, and the final prediction is based on the majority vote of all the trees. This approach reduces overfitting and increases robustness.
